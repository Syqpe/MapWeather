Project architecture inspired by - [FSD](https://feature-sliced.design/)

## Development

### How to run, and build?

1. Install:

    ```sh
    brew install nvm

    nvm install 14

    npm install
    ```

2. Run

    ```sh
    npm run dev
    ```

3. Build

    ```sh
    npm run build

    ```
